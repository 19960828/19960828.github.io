Windows 系统安装 Nginx 环境

搭建环境是最基本的操作，结合官方的文档，很简单的就安装好了。

下载

官网下载地址：https://nginx.org/en/download.html
下载版本名称：nginx/Windows-1.15.9

解压安装

  • 将下载下来的 nginx-1.15.9.zip 文件进行解压，将减压后的文件放在你要安装的目录下；我这边的安装目录是在”D:\Program Files”目录下
  • 按下 Win + R 组合键在弹框处输入 cmd 进入命令提示符操作界面，操作步骤如下图代码所示：

如果是出现一下内容，即启动成功（参数不一定要一样哦，有两条记录显示就说明启动成功了）

最后

更多的环境搭建内容可以见官网的 nginx for Windows 的安装教程。
